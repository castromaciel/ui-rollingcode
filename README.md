## Download 

With npm
```
npm install ui-rc-components
```

or install with yarn
```
yarn add ui-rc-components
```

## Usage

```ts
import { ExampleComponent } from 'ui-rc-components';
```

```tsx
<ExampleComponent 
  label='Example label'
/>
```
